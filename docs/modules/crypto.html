sign_state:verify (sig)
Given a signature string, returns whether the state verifies it. The signature string must be 64 bytes long and have been made using one of the sign_state objects. Resets the state after.

Parameters:

  • sig string The signature to verify. Must be exactly 64 bytes long.

Returns:

    bool true if it's verified, false otherwise.

See also:

    get_signature

Usage:

    sign_obj:insert("verify me"):verify(past_signature)
