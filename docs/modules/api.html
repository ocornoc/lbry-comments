API

ping ()
Returns the string "pong". This function is hyper-optimized and uses a lot of very high-level computer science techniques in order to produce the output it does.

Returns:

    string "pong"

Usage:

    {"jsonrpc": "2.0", "method": "ping", "id": 1} -> [server]
 [server] -> {"jsonrpc": "2.0", "id": 1, "result": "pong"}
status ()
Returns the status and versions of the server components.

Returns:

    table status

    status.is_running is a boolean, always true.

    status.is_db_running is a boolean, describing whether the database is currently running.

    status.api_version is a string, representing the SemVer 2.0.0 version of the API.

    status.db_version is a string, representing the SemVer 2.0.0 version of the database.

Usage:

    {"jsonrpc": "2.0", "method": "status", "id": 1} -> [server]
[server] -> {"jsonrpc": "2.0", "id": 1, "result": {
	"is_running": true,
	"is_db_running": true,
	"api_version": "0.0.0",
	"db_version": "0.0.3"
}}
