How to use

You must use the file start as it supplies the server with some necessary global variables.

Run ./start -h for more details.

Dependencies

  • libsodium
  • OpenResty
  • LuaSQL (using SQLite3)
  • busted (only for debug)
  • LDoc (only for documentation gen)

LuaSQL and Busted must be installed using appropriate OPM packages or manually installed in OpenResty's tree. LDoc doesn't have to be installed in the tree.
