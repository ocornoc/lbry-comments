Busted support

If you have busted installed, you can run busted from the project directory in order to run the tests.

Dependencies

  • libsodium
  • OpenResty
  • luajit >= 2.0.0 (included in OpenResty)
  • lua-cjson (included in OpenResty)
  • luasocket
  • LuaFileSystem
  • LuaSQL (using SQLite3)
  • busted (only for debug)
